# Churn SaaS Project

This repository contains a churn prediction SaaS project: data preprocessing, model training, and a web service to serve predictions.

## Contents

- Data processing scripts
- Model training and evaluation
- Web app for serving predictions

## Getting started

1. Install dependencies (see project files for language-specific requirements).
2. Run preprocessing and training scripts to prepare the model.
3. Start the web service to serve predictions.
